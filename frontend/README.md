# Jupiter IT Services

**Tech Stack:** React.js | Java Spring Boot | SQL Database  

This project is an **application** developed to manage internal business operations. The frontend is React.js, the backend is Spring Boot, and SQL is used for database management.

## Features
- Responsive frontend with React.js  
- Backend REST APIs with Spring Boot  
- SQL database for storing employee, client, and project data  
- Authentication and role-based access  
- CRUD operations for internal modules  

## How to Run
1. Clone the repo  
2. Backend: Run Spring Boot project (`mvn spring-boot:run`)  
3. Frontend: Navigate to React folder, run `npm install` then `npm start`  
4. Access frontend at `http://localhost:3000`
